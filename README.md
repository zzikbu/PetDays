## 🐶 펫로그

## 프로젝트 소개

1500만 반려동물 인구를 위한 반려동물 정보, 성장 일기, 산책 기록, 진료 기록을 관리할 수 있는 모바일 애플리케이션입니다.

## 개발 환경

- Flutter (version 3.22.3)
- Dart (version 3.4.4)
- Android Studio (version 2024.1)
- Xcode (version 15.4)

## 개발기간

2024.08 ~
